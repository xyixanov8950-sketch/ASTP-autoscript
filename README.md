Инструкция использования данного скрипта
Версия на русском
Создатель: dvlk2

Скрипт используется для быстрой замены кода в файле с расширением .py
Подходит для Вайб Кодеров — тех, кто пишет код с помощью Искусственного Интеллекта

Окошко скрипта выполнено в стиле Windows XP

Как использовать:
0. Скачайте ехе файл
1. Перенесите ваш .py файл в окно скрипта — через кнопку «Обзор» или напрямую из проводника
2. Убедитесь, что в буфере обмена уже есть новый код, которым нужно заменить старый
3. Нажмите «Собрать всё» (или используйте горячую клавишу Ctrl+Alt+Shift+P)
4. Скрипт сам переименует .py → .txt, откроет в Блокноте, вставит новый код, сохранит, закроет, переименует обратно в .py

Необходимые библиотеки (установка одной командой):
pip install pyautogui keyboard pillow requests

Готово! ✨

Instructions for using this script
English version
Creator: dvlk2

This script is designed for quickly replacing code inside a .py file
Perfect for Vibe Coders — those who write code with the help of Artificial Intelligence

The script window is styled like Windows XP

How to use:
0. Download EXE files
1. Drag and drop your .py file into the script window — using the "Browse" button or directly from File Explorer
2. Make sure the new code is already copied to your clipboard
3. Press "Assemble Everything" (or use the hotkey Ctrl+Alt+Shift+P)
4. The script will automatically rename .py → .txt, open it in Notepad, paste the new code, save, close, rename it back to .py

Required libraries (install with one command):
pip install pyautogui keyboard pillow requests

Done! ✨
